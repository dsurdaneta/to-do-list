To add a task:

  1. Click/Touch the Add Task button
  2. Fill the form.
  3. Click/Touch the Add button.
  4. Repeat as many times you want
  5. Click/Touch the Done button

Move a task to another category:

  1. Just dragg the task down (or up) from the current category box to the corresponding category box.

Clear or delete all the task:

  1. Click/Touch the Clear All button

Delete a single task:

  1. When you are dragging a task, a delete box will appear next to the Clear All button.
  2. Drop the task in that magic box.
